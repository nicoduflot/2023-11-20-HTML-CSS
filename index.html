Introduction

Balises générales de contenu

Généralement, tous les contenus de type texte sont a mettre dans des balises de type paragraphe. Il n'existe pas que la balise <p></p>

Il existe deux type de balise de contenu :

  • Block :
    La balise prend tout l'espace en largeur, et ne peut pas contenir d'autres balise de contenu de type block, sauf si on le contre-indique en CSS.
  • Inline :
    Ces balises "entourent" le texte, sans générer (sauf contre-indication) de marge, d'espace, etc.

Balises sur le texte

Il existe des balise de type inline pour modifier l'apparence du texte.

Par exemple, la balise usuelle pour du gras, on utilise <b>, mais si le texte mis en gras a un importance sur la pertinence au niveau du SEO, on utilisera <strong>. C'est important. Le même type de comportement est a noté entre l'italique <i> et l'emphase <em>.

Autrefois, pour changer la couleur, mapolice et la taille d'un extraît de texte, on utilisait la la balise <font>. Un exemple, maintenant on créera plutôt une classe que l'on appelera dans lune balise <span> comme ceci.
