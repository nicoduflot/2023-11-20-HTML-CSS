Introduction

Balises générales de contenu

Généralement, tous les contenus de type texte sont a mettre dans des balises de type paragraphe. Il n'existe pas que la balise <p></p>

Il existe deux type de balise de contenu :

  • Block :
    La balise prend tout l'espace en largeur, et ne peut pas contenir d'autres balise de contenu de type block, sauf si on le contre-indique en CSS.
  • Inline :
    Ces balises "entourent" le texte, sans générer (sauf contre-indication) de marge, d'espace, etc.

Media image

Par défaut, on utilise la balise img, elle a pour attribut principal src (source), qui va chercher l'image et l'affiche sur la page. Ensuite, pour tout l'accessibilité, on ajoute la balise alt, qui le texte alternatif de description de l'image, qui a deux utilité : décrire l'image via le lecteur d'acran pour les personnes aveugles ou malvoyantes, et aussi a afficher un texte si l'image est soit "casssée" ou absente.

img src="./images/monimage.png" alt="Image d'exemple de la balise img" /

Avec HTML5, une balise conteneur existe pour apporter un meilleur référencement aux images ayant un contexte particulier.

Il est possible de gérer un adaptatif évolutif, au lieu d'avoir le CSS qui réduit une grande image, on peut choisir d'afficher une image d'une certaine taille selon la largeur d'écran disponible. Avec des images moins lourde pour chaque taille, on améliore la rapidité de chargement.

l'attribut srcset n'est pas uniquement utilisé par source /, il peut être utilisé avec img / mais nécéssite un attributs supplémentaire : sizes.
